# Задачи за практикум

## Задача 1
Да се дефинират операторите >> и << за следния клас

```C++
class gcd {
    private:
        int x, y;

    gcd(int a, int b) {
        x = a;
        y = b;
    }

    int value() const {
        if(y == 0) {
            return x;
        } else {
            gcd t(y, x % y);
            return t.value();
        }
    }
};
```
намиращ най-големия общ делител на две цели неотрицателни числа, поне едното от които е различно от 0;
